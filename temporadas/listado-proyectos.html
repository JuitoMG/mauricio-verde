04 - La librería de segunda mano

Dos programas, "11-libreria.py" conectado a una base de datos de libros, "libreria.csv", que permite registrar libros nuevos, modificar los ya existentes, buscar por título o autor y mostrar todo el listado ordenado alfabéticamente por autor.

El otro programa, "12-restaurador_libros.py", está hecho para restaurar un archivo rellenando campos vacíos o mal escritos de un archivo corrupto, "librosant.csv", generando un archivo nuevo llamado "libros_restaurados.csv".
